# OrderFlow 🚀

OrderFlow is a MERN-based web app that helps local vendors manage and track orders efficiently.

## Features
- Add and manage orders
- Status tracking (Pending, Delivered)
- Search & filter
- WhatsApp integration

## Tech Stack
- React + Tailwind
- Node + Express
- MongoDB
